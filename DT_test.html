Kατάληψη 10 στρατηγικών στόχων


Εκφώνηση

Σε μια πολεμική επιχείρηση εσείς και ο αντίπαλος στρατηγός επιδιώκετε να καταλάβετε 10 στρατηγικούς στόχους, διαφορετικής μεταξύ τους αξίας. Συγκεκριμένα ο 1ος στόχος έχει αξία 1 βαθμό, ο 2ος 2 βαθμούς, κ.ο.κ. έως το 10. Και οι δύο έχετε από 100 στρατιώτες στη διάθεσή σας, τους οποίους πρέπει να μοιράσετε στους 10 στόχους, ώστε να προσπαθήσουν να τους καταλάβουν. Όποιος στείλει περισσότερους στρατιώτες σε ένα συγκεκριμένο στόχο τον καταλαμβάνει και κερδίζει τους βαθμούς που του αναλογούν. Αν έχει σταλεί ο ίδιος αριθμός στρατιωτών και από τους δύο αντίπαλους, οι βαθμοί του στόχου μοιράζονται στα δύο. Προφανώς, δεν γνωρίζετε την κατανομή των δυνάμεων του εχθρού ώσπου να ξεκινήσει η μάχη. Όποιος κερδίσει τους περισσότερους βαθμούς κερδίζει τον πόλεμο. Σχετική δημοσίευση στο facebook.


library(DT)
library(dplyr)
library(purrr)
library(datacomb)

data1 <- read.csv("GAME-Sheet1.csv",header = FALSE)
data1 <- data1[,1:11]
names(data1) <- c("Player", paste0("X",1:10))
data1$Player <- sub("_"," ",data1$Player)
data <- data1[1:12,]

resolve_conflicts <- function(dat){
  da <- dat %>% mutate(N=0,I=0,H=0)
  comb <- combn(nrow(dat),2)
  dat <- as.matrix(da[,2:14])
  
  apply(comb, 2, function(x){
    points1 <- sum(map_dbl(dat[x[1],1:10] - dat[x[2],1:10],
                         ~if(.x>0) 1 else if (.x<0) 0 else .5) * 1:10)
    points2 <- 55 - points1
    if(points1>points2) {
      dat[x[1],"N"] <<- dat[x[1],"N"] + 1
      dat[x[2],"H"] <<- dat[x[2],"H"] + 1
    } else if(points2>points1){
      dat[x[2],"N"] <<- dat[x[2],"N"] + 1
      dat[x[1],"H"] <<- dat[x[1],"H"] + 1
    } else {
      dat[x[2],"I"] <<- dat[x[2],"I"] + 1
      dat[x[1],"I"] <<- dat[x[1],"I"] + 1
    }
  })
  
  dat <- data.frame(Player=da[,1],dat)
  return(dat %>% arrange(desc(N),H))
}

Βασικό δείγμα

Στη θεωρία παιγνίων, αυτό το είδος προβλήματος είναι γνωστό ως Colonel Blotto. Για τα παιχνίδια Blotto είναι υπολογιστικά δυνατό να βρεθεί ισορροπία Nash, αλλά δεν είναι πάντα απλό. Βλ. Computing Equilibria of Blotto and Other Games.

Ο παίχτης που τερμάτισε 1ος περιέγραψε τη στρατηγική του με τα παρακάτω (την ίδια στρατηγική έχει ακολουθήσει και ο παίχτης που τερμάτισε 3ος):

Υποψιάζομαι ότι πολλές στρατηγικές θα στέλνουν 0 ή 1 στρατιώτες σε κάποιους στόχους και θα επικεντρώνουν την επίθεσή τους σε στόχους με άθροισμα 28 βαθμούς. Στις περιπτώσεις αυτές θα έχουμε κάποιο συνδυασμό μηδενικών τιμών και τιμών πάνω-κάτω σας αυτές: 4, 7, 11, 15, 18, 22, 25, 29, 33, 36. Άλλες πάλι θα κατανέμουν τους στρατιώτες ανάλογα με τους βαθμούς των στόχων. Τότε οι κατανομές θα μοιάζουν με αυτή: 2, 4, 5, 7, 9, 11, 13, 15, 16, 18.

Γενικά, θα νικάω έναν αντίπαλο αν καταφέρω να κερδίζω στόχους έχοντας μόνο ελάχιστα περισσότερους στρατιώτες από αυτόν και χάνοντας εκεί που έχω πολύ λιγότερους από αυτόν.

Το σχέδιό μου στοχεύει σε ουσιαστικά ανυπεράσπιστους στόχους με βαθμούς 6, 9 και 10, σε ανάλογα κατανεμημένες δυνάμεις για τους στόχους 1, 2, 3, 4 και 5 και σε εστιασμένες επιθέσεις στους στόχους 7 και 8.
